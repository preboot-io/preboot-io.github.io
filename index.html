PPreboot

Build SaaS Apps Faster

A comprehensive framework for Java developers building multi-tenant applications with Spring Boot

Why Preboot?

Stop reimplementing the same patterns. Focus on your business logic while we take care of the infrastructure.

Multi-Tenant Authentication

Secure user management with built-in tenant isolation. Includes JWT, OAuth2 support, and role-based access control out of the box.

Dynamic Data Filtering

Automatic query scoping based on active tenant security context. Perfect data isolation without changing your business logic.

Observability Ready

Logs, metrics, and health checks ready for production. Monitor your application with ease using standard tools and practices.

Event-Driven Architecture

Built-in event bus, task queues, and message handling. Scale your application with asynchronous communication patterns.

CQRS & Clean Architecture

Command Query Responsibility Segregation implementation with clean separation of concerns for better maintainability.

Modular Components

Pick what you need. Each feature is modular and can be used independently, while still working seamlessly together.

How It Works

Getting started with Preboot is simple. Follow these steps to accelerate your SaaS development.

1
Install the Dependency
Add Preboot to your Spring Boot project using Maven or Gradle. The core library is lightweight and has minimal dependencies.
2
Configure Your Application
Use our simple builder pattern to enable the features you need. Configure multi-tenancy, authentication, and other components with minimal code.
3
Build Your Business Logic
Focus on what makes your application unique. Use our CQRS implementation to build clean, maintainable business logic without the boilerplate.
4
Deploy with Confidence
With built-in observability and production-ready features, your application is ready for the real world. Scale with ease as your user base grows.

Code Examples

See how Preboot simplifies your development process with clean, intuitive APIs.

@Configuration
public class PrebootConfig {
    @Bean
    public PreBootKit preBootKit() {
        return PreBootKit.builder()
            .withMultiTenancy()
            .withAuthentication()
            .withEventBus()
            .withTaskQueue()
            .build();
    }
}
@RestController
@RequestMapping("/api/customers")
public class CustomerController {
    
    private final CustomerService customerService;
    
    // Constructor injection...
    
    @GetMapping
    public List<CustomerDto> getAllCustomers() {
        // Tenant filtering automatically applied
        return customerService.findAll();
    }
    
    @PostMapping
    public ResponseEntity<CustomerDto> createCustomer(@RequestBody CreateCustomerRequest request) {
        // Customer automatically associated with current tenant
        CustomerDto created = customerService.create(request);
        return ResponseEntity.created(URI.create("/api/customers/" + created.getId()))
            .body(created);
    }
}
// Command
@Value
public class CreateProductCommand implements Command {
    private String name;
    private String description;
    private BigDecimal price;
}

// Command Handler
@Component
class CreateProductCommandHandler implements CommandHandler<CreateProductCommand> {
    private final ProductRepository repository;
    private final EventBus eventBus;
    
    // Constructor injection...
    
    @Override
    public void handle(CreateProductCommand command) {
        Product product = new Product(
            UUID.randomUUID(),
            command.getName(),
            command.getDescription(),
            command.getPrice()
        );
        
        repository.save(product);
        eventBus.publish(new ProductCreatedEvent(product.getId()));
    }
}
