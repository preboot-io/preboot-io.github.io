Enterprise Patterns Made Accessible

Production-ready patterns learned from scaling Atlassian. Now available to every Java developer.

Multi-Tenant Authentication

Complete multi-tenant user management with JWT support, role-based access control, and automatic tenant context - all with minimal configuration.

Secure Data Access

A simple @Tenant annotation provides automatic tenant isolation on all your queries. Build complex, type-safe dynamic queries with built-in security filtering.

Observability Ready

Built-in monitoring for slow method calls, detailed error tracking, and health checks. Easily identify performance bottlenecks even in development.

Event-Driven Architecture

Complete event system with synchronous and asynchronous handling, persistent task queues with retry logic, and type-safe events for building reactive applications.

Full-Stack SaaS Solution

Backend and frontend libraries optimized for SaaS development with React components for auth flows, data tables, and form handling - all integrated with tenant context.

Modular Architecture

Pick and choose from modular components that work together seamlessly. Use just what you need with minimal dependencies, or adopt the entire framework for maximum productivity.

{% highlight java %} // Enterprise-grade multi-tenancy in 3 annotations @Table("documents") @SecureAccess( read = @AccessRule(roles = {"VIEWER", "EDITOR", "ADMIN"}), write = @AccessRule(roles = {"EDITOR", "ADMIN"}) ) public class Document { @Id private Long id; @Tenant // Automatic tenant filtering - never see other tenants' data private UUID tenantId; private String title; private String content; @Version private Long version; // Optimistic locking built-in } {% endhighlight %}
{% highlight java %} // Zero-config security - tenant isolation automatic public interface DocumentRepository extends SecureRepository { // Find methods automatically filtered by tenant Optional findByTitle(String title); // Custom queries with automatic security @Query("SELECT * FROM documents WHERE status = :status") List findByStatus(String status); // Tenant filter added automatically } // Dynamic filtering with type-safety @Repository public class DocumentService { public Page search(String titlePattern, String status) { return documentRepository.findAll( SearchParams.criteria( FilterCriteria.like("title", titlePattern), FilterCriteria.eq("status", status) ).sortBy("createdAt").build() ); // Tenant security automatic } } {% endhighlight %}
{% highlight java %} // Event-driven architecture from Atlassian playbook public record DocumentCreatedEvent( UUID documentId, UUID tenantId, String title ) {} @Component public class DocumentEventHandler { @EventHandler // Async by default, with retry logic public void handleDocumentCreated(DocumentCreatedEvent event) { // Send notification - tenant-aware automatically notificationApi.sendEmail(EmailParam.builder() .template("document-notification") .addParam("title", event.title()) .build()); } } // Background tasks with enterprise reliability @Service public class DocumentService { public void processLargeDocument(UUID documentId) { // Persistent queue with retries and dead letter handling taskPublisher.publishTask(new ProcessDocumentTask(documentId)); } } {% endhighlight %}
